# CityU FusionX: Decentralized Credit Scoring System

## Overview
An inclusive, explainable credit scoring system designed for individuals with limited financial history. This prototype simulates a decentralized ledger and AI-driven risk assessment to ensure data privacy, fairness, and trust.

## Tech Stack (Prototype Phase)
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Simulated using browser `localStorage`
* **Blockchain:** Simulated by saving hashed transaction data locally
* **AI:** Rule-based scoring algorithm generating explainable risk profiles

## Setup Instructions
1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. No external database or server installation is required for this prototype.
